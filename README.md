## 个人自律自治，一站式个人管理，助力成长的应用；后期目标私域共治。
> 目前发布版本：MacOs V1.1.0 基础完善款；Window版正在配置调试中， 敬请期待；

技术栈：Electron + React + Ant Design + Webpack

### 现已有功能：

1. 待办汇集；
2. Markdown笔记整理；
3. 目标打卡；
4. PDF编辑阅读等；

### 特点：
- 目录不限制文件类型；
- 待办可以产出周期报告；
- 笔记由slate打造，所见即所得；
- 分层式目标打卡；
- 支持pdf阅读与编辑；
